# Create: Pocket Factory 说明文档

## 1. 项目定位

Create: Pocket Factory 是一个面向机械动力自动化玩家的独立维度工厂模组，设计它的目的是为了降低显卡占用

核心思想：

1. 将高渲染负担的自动化机器迁移到单独维度。
2. 玩家离开工厂后，机器继续运行，仍消耗TPS，但不再占用常驻活动区域的渲染预算，可显著降低FPS压力。
3. 提供一些便于跨纬度传输流体、物品、应力的手段。

## 2. 设定说明

1. 玩家可以通过入口进入新维度中的一个白灰棋盘格装饰的盒子内，该盒子即为“口袋工厂”。
2. 口袋工厂尽可能保持少量占用区块，且其中始终常加载，以让玩家的机械设备始终工作。
3. 玩家从工厂返回时，优先回到进入前记录的位置；若原位置无效，再回主世界出生点。
4. 玩家进入工厂时，优先传送到查找到的第一个地毯上，其次查找一个允许站立的空位。
5. 玩家可以消耗物品扩展工厂的尺寸。
6. 工厂维度始终提供较为明亮的环境，工厂盒子外壳有发光方块搭建，避免玩家在光影或非光影环境下进入口袋工厂后视野受限。
7. 玩家可以通过核心右键某些create物块来将它们转化为“关联版”用于跨纬度自动化传输。

## 3. 物品说明

### 工厂核心

作为工厂入口的合成材料，以及大部分关联物品生成的方式：

1. 手持右键口袋工厂方块A(也就是口袋工厂内的墙壁或天花板地板上的白色方块)，可用于触发工厂扩容。横向扩容消耗1个核心；垂直扩容固定为单次1格。
2. 手持右键两个物品保险库，可将两组多方块结构转换为关联物品保险库。
3. 手持右键两个流体储罐，可将两组多方块结构转换为关联流体储罐。
4. 手持右键两个垂直溜槽，可建立一条关联溜槽绑定。
5. 手持右键两个离合器，可建立一条关联离合器绑定。
6. 手持右键两个动力泵，可建立一条关联动力泵绑定。
7. 可用于合成工厂入口。

### 工厂入口

- 作为一般玩家进入口袋工厂的途径，放置下来右键它即可传送进入口袋工厂。
- 该方块中有一个微缩的口袋工厂中心模型，可能会无法渲染一些方块，但大致能让玩家看懂这个工厂对应的是哪一个。
- 每个入口放置后就会生成一个新的口袋工厂用于绑定，拆除后会保持绑定，重新放置即可再次进入。
- 如果工厂入口因某些原因丢失，可以使用指令来恢复一个对应的入口，又或者真的就无法再次进入这个工厂了（假如你没开作弊的话）。

### 关联类物品

- 不能制作，只能通过手持核心转化对应的普通版物品。
- 挖掉后掉落对应的普通版物品，被关联的另一侧转化为原版物块。
- 一对一，容器类共享容量，物流类交叉物流。
- 外观与原版物品一致。
- 允许跨维度关联
- 由于作者能力问题，无法提供关联传送带、锁链传动轮等会引入物品网络的物流设备。

#### 物品保险库

- 手持工厂核心右键两个物品保险库来转换成关联物品保险库。
- 整个结构都会被转换。
- 关联的物品保险库中内容会共享，总容量为二者容量上限相加。
- 当任意一个关联物品保险库方块被破坏时，当前的多方块结构会被摧毁掉落，被关联的另一侧将会还原回原版物品保险库结构。

#### 流体储罐

- 手持工厂核心右键两个流体储罐来转换成关联流体储罐。
- 整个结构都会被转换。
- 关联的流体储罐中内容会共享，总容量为二者容量上限相加。
- 当任意一个关联流体储罐方块被破坏时，当前的多方块结构会被摧毁掉落，被关联的另一侧将会还原回原版流体储罐结构。

#### 溜槽

- 手持工厂核心右键两个溜槽（不能是倾斜溜槽）来转换成关联溜槽。
- 关联的溜槽A和B之间的尾部会互换，进入A的物品会从B下方掉落，气流从B下方吹上会从A上方继续向上吹。
- 当任意一个关联溜槽被破坏时，被关联的另一侧将会还原回原版溜槽。

#### 动力泵

- 手持工厂核心右键两个动力泵来转换成关联动力泵。
- 关联的动力泵A和B之间的尾部会互换，进入A的流体会从B尾部被泵出。
- 由于性能问题，泵出的液体无法被输出到空气成为流体方块。
- 当任意一个关联动力泵被破坏时，被关联的另一侧将会还原回原版动力泵。

#### 离合器

- 手持工厂核心右键两个离合器来转换成关联离合器。
- 关联离合器拥有两种状态，分别属于两侧同信号和两侧不同信号（也就是是否被红石激活）。
- 当两侧均为激活或未激活时（两侧同信号），处于直通状态；两侧只有一侧激活时，处于交叉状态。
- 直通状态下的关联离合器与传动杆行为一致
- 交叉状态下关联的离合器A和B之间的尾部会互换，传入A头部的应力和转速会从B尾部输出、传入B头部的应力和转速会从A尾部输出、传入A尾部的应力和转速会从B头部输出、传入B尾部的应力和转速会从A头部输出。
- 无法当交叉相连的一条线路中已经有一个输入端，另一侧将被固定为输出端（如A头部已经是输入端，那么B尾部将无法再输入）。
- 当任意一个关联离合器被破坏时，被关联的另一侧将会还原回原版离合器。
